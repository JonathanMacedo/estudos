Numero + . = Inseri uma lista numerada. Ex: 
1. carro 2 
2.  carro 1 

Lista demarcada = Inseri uma lista demarcada. Ex:
<br>
* Teste

  * Teste

- [X] Fazer o front
- [X] Fazer o back
